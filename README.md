# Graphs library in Python 3.*

This is a Python 3.* for implementing graphs

## General examples

### Graph creation

Syntax :
```python
g = Graph(listofnodes, listofarcs=[])
```

You can create a simple graph like this :

```python
g1 = Graph(["a", "b", "c"], [["a", "b"], ["b", "c"], ["c", "a"]])
```
